### GEEK
#### 在目录themes下存放

下载说明

进入 Hexo -> themes 目录下

终端执行以下命令

``` bash
$  git clone https://github.com/sanjinhub/hexo-theme-geek.git geek 
```
修改 hexo -> _config.yml 的 theme 为 geek

More info: [GEEK](https://github.com/sanjinhub/hexo-theme-geek/)
